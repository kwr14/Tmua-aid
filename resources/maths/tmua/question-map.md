# TMUA Question Map

This note maps the TMUA paper library by topic so you can go straight to the most useful questions for targeted practice.

Use it alongside the [Topic Drill Plan](drill-plan.md).

## How To Use This Map

- If a topic feels weak, start with the `Best Starting Set`.
- Then try the `Best Paper` for a deeper block of practice.
- Use the `Stretch` questions once the method feels familiar.
- Review the worked note after attempting the question yourself first.

## Suggested Attempt Order

If you want a fast route from weak topic to stronger confidence:

1. Do the `Best Starting Set` for one topic.
2. Review the worked solutions and extract the method trigger.
3. Attempt the `Best Paper` for that topic under light time pressure.
4. Finish with one `Stretch` question.
5. Return to the [Topic Drill Plan](drill-plan.md) and repeat for the next weak area.
