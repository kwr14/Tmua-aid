# TMUA One-Page Cheat Sheet

This is the short version of the TMUA study pack.

Use it the day before a paper or in the final 10 minutes before you start practice.

## First Questions To Ask

When you open a question, ask:

1. Is this really a quadratic in disguise?
2. Would a sketch make this obvious?
3. Is this about number of solutions, not the exact solutions?
4. Can I use structure or symmetry instead of brute force?
5. Do the answer options suggest the method?

## Algebra Essentials

- Factor before you expand.
- If the expression repeats, substitute.
- Complete the square for minima, maxima, graph shifts, and circles.
- Use the discriminant for intersections, tangency, and root counting.
- Watch sign changes after multiplying or dividing inequalities.

Most useful patterns:

- `a^2 - b^2 = (a-b)(a+b)`
- `x^2 + bx = (x + b/2)^2 - (b/2)^2`
- `b^2 - 4ac`

## Graph Thinking

- A graph can answer “how many solutions?” faster than algebra.
- Turning points control how many times a curve can meet a line.
- Transformations matter:
- `f(x-c)` right
- `f(x+c)` left
- `f(x)+k` up
- `af(x)` vertical stretch
- `f(kx)` horizontal scale by `1/k`

Key habit:

- translate equations into intersections

## Sequences And Series

- Arithmetic: constant difference
- Geometric: constant ratio
- Infinite geometric sum exists only when `|r| < 1`

Core formulas:

- arithmetic sum: `S_n = n/2 (first + last)`
- geometric sum: `S_n = a(1-r^n)/(1-r)`, `r != 1`
- infinite geometric sum: `S = a/(1-r)`, `|r| < 1`

## Trigonometry

Know these cold:

- `sin^2 x + cos^2 x = 1`
- `tan x = sin x / cos x`
- exact values for `0`, `30`, `45`, `60`, `90`

Workflow:

1. Rewrite everything using `sin` and `cos` if needed.
2. Factor if possible.
3. Solve each case.
4. Count solutions on the stated interval carefully.

## Geometry And Coordinate Geometry

- Sketch first.
- Similar triangles are everywhere.
- A radius is perpendicular to a tangent.
- Complete the square to identify a circle.
- For maximum or minimum distance, think geometrically before calculating.

Area tools:

- `1/2 * base * height`
- `1/2 ab sin C`
- subtract from a larger simple shape
- use ratios if exact area is unnecessary

## Calculus Essentials

- A derivative tells you where turning points happen.
- A definite integral is often easier as area than as algebra.
- If the graph is familiar, use geometry.
- Break awkward integrals into smaller intervals if the question gives a pattern.

## Number And Logic

- Translate divisibility into prime factors.
- Check parity early.
- Use a counterexample if a statement claims to be always true.
- Restate wordy definitions in simple mathematical language before solving.

## Multiple-Choice Strategy

- Eliminate by sign, size, parity, or domain before doing full work.
- Compare options if exact calculation looks messy.
- If all options are close, your method must be precise.
- If one option is impossible by rough reasoning, remove it immediately.

## Common Fast Routes

- line meets parabola -> discriminant
- ugly algebra -> substitution
- odd-looking integral -> area
- circle equation not in standard form -> complete the square
- repeated angle geometry -> similar triangles
- “always true?” -> search for a counterexample
- infinite sum -> check `|r| < 1` first

## Final Reminders

- simplify early
- do not expand blindly
- count solutions carefully
- keep the interval in view
- use the graph whenever possible
- the shortest correct method is usually the TMUA method
